## Protocol introduction
**Define**
```
[protocol_name;4][id;12][code;1][message;_]
```
**Name**: `MOYU`
**Code**
|code|command|
|----|-------|
|`0`|search message|
|`1`|connect request|
|`2`|connect reply|
|`3`|send or receive message|
